# Mutlu Yavuz - GitHub Pages Kişisel Site

Bu paket GitHub Pages üzerinde yayınlanabilecek modern bir kişisel site taslağıdır.

## Kurulum

1. GitHub'da yeni bir repo aç.
   - Kişisel ana site istiyorsan repo adı: `kullaniciadi.github.io`
2. Bu dosyaları repoya yükle.
3. Birkaç dakika içinde şu adreste yayınlanır:
   - `https://kullaniciadi.github.io`

## Düzenlenecek Alanlar

### index.html içinde:
- Footer'daki GitHub, LinkedIn ve e-posta linkleri
- Proje açıklamaları
- Hakkımda metni

### styles.css içinde:
- Renkler
- Kart stilleri
- Buton ve tipografi ayarları
